# Instagram 크롤링

> Some scrapy spiders useful to crawl instagram posts using public APIS (No TOKEN)

## Requirements
- Python
- Scrapy

## Spiders
- hashtag (crawl all the post given a hashtag)

## 사용

```python
scrapy crawl hashtag
```
Use the -L INFO to avoid a lot of debug messages
